# Webhook Instructions

 
 Author Garrett Smith

This incorporates Webex Incoming Webhook as an alternative to PagerDuty notifications through it's respective bot.
 Webex Webhook features available to developers on Webex Development Page.

 Step 1. Create a Webex account and login into Webex Developers page with your profile.

 Step 2. Click create a Webex app.

 Step 3. Create a Webex Space. This space will be associated with your webhook and can be done so by adding the
 webhook to the space in the dropdown menu while setting up the Webhook. IMPORTANT: You must have the Webex Space
 created prior to creating the webhook, additionally the webhook has to have a specified name

 Step 4. Link your Webex Space with the Webex Webhook available on the Webex Developers Site.
